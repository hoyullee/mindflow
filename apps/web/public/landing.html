학습 노트

강의와 책의 핵심을 구조로 묶어 오래 기억에 남게 정리해요.

프로젝트 기획

목표에서 할 일로 가지를 뻗어 큰 그림과 세부 계획을 함께 세워요.

회의록

논의의 흐름을 실시간으로 정리하고 참석자와 바로 공유해요.

글쓰기 개요

글의 뼈대를 먼저 짜고 살을 붙여, 논리의 흐름을 놓치지 않아요.

의사결정

선택지와 근거를 한눈에 늘어놓고 비교해 더 나은 결정을 내려요.

3

저장하고 공유해요

자동으로 저장되고, 완성한 맵은 이미지나 파일로 내보내 함께 나눠요.

자유로운 마인드맵

노드·도형·메모·연결선·영역으로 생각을 자유롭게 배치하고, 색과 스타일로 구조를 드러내세요.

어디서나 이어서

웹과 모바일(PWA) 어디서든 열립니다. 자동 저장과 기기 간 동기화로 흐름이 끊기지 않아요.

함께, 그리고 내보내기

궁금한 점이 있으신가요?

정말 무료인가요?

네. 별도의 설치나 결제 없이 지금 바로 시작할 수 있어요.

설치해야 하나요?

아니요. 웹 브라우저만 있으면 됩니다. 모바일에서는 홈 화면에 추가하면 앱처럼 전체 화면으로 사용할 수 있어요.

만든 맵은 안전하게 보관되나요?

편집하는 동안 자동으로 저장되고, 로그인하면 여러 기기에서 같은 맵을 이어서 볼 수 있어요.

다른 곳으로 내보낼 수 있나요?

완성한 맵을 PNG 이미지, Markdown, JSON 파일로 내보낼 수 있어 다른 문서나 도구로 바로 옮길 수 있어요.
